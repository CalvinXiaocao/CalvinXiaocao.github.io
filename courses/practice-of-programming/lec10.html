Standard Template Library

使用模版的程序设计方法，将一些数据结构（链表，数组，二叉树）和算法（排序，查找）写成模版

STL: 一些常用数据结构和算法的模版的集合

STL中的基本概念

  • 容器：容纳各种数据类型的数据结构
  • 迭代器：Iterator 可依次存取容器中元素的工具
    • 普通C++指针就是一种迭代器
  • 算法：用来操作元素的函数模版

容器

存放各种类型的数据 1. 顺序容器 vector, deque, list 2. 关联容器、有序容器 set, multiset, map, multimap 3. 容器适配器 stack, queue, priority_queue

1和2统一称为第一类容器

对象被插入容器时，插入的是对象的一个复制

放入对象所属的类，应该实现==和<运算符

顺序容器简介

vector

动态数组，随机存取任何元素的复杂度是\(O(1)\)，在尾部增删元素具有较高的性能 #### deque 动态数组，随机存取任何元素的复杂度是\(O(1)\)（次于vector），在两端增删元素具有较高的性能 ### list 双向链表，在任何位置增删元素都能在常数时间内完成，不支持随机存取

关联容器简介

set/multiset

集合，set不允许相同元素，multiset中允许存在相同的元素 #### map/multimap 映射，map中存放的是成对的key/value，并根据key对元素进行排序，可快速根据key检索元素

插入和检索时间都是\(O(\log n)\)

容器适配器

stack

栈，后进先出 #### queue 队列，先进先出 #### priority_queue 优先队列，最高优先级元素第一个出列

Iterator 迭代器

用于指向第一类容器中的元素，有const和非const两种

通过迭代器可以读取它指向的元素

通过非const迭代器还可以修改其指向的元素

迭代器可以执行++操作，指向容器的下一个元素

如果迭代器到达了某一个容器中的最后一个元素的后面，则past-the-end

#include <vector>
#include <iostream>
using namespace std;
int main() {
    vector<int> v;
    v.push_back(1);
    v.push_back(2);
    v.push_back(3);
    v.push_back(4);
    vector<int>::const_iterator i;
    for (i = v.begin(); i != v.end(); i++) {
        cout << *i << endl;
    }

    vector<int>::reverse_iterator r;
    for (r = v.rbegin(); r != v.rend(); r++) {
        cout << *r << endl;
    }

    vector<int>::iterator j;
    for (j = v.begin(); j != v.end(); j++) {
        *j = 100;
        cout << *j << endl;
    }
}

容器与迭代器

容器的迭代器按功能分为五种

全部：++p, p++ * 输入 p, p = p1, p == p1, p != p1 输出 p, p = p1 正向 上面全部 * 双向 上面全部, –p, p– * 随机访问 p += i, p -= i，<

vector和deque是随机迭代器，list/set/map是双向迭代器

算法简介

70多种标准算法

算法就是一个个函数模版，通过迭代器操作容器中的元素

许多算法需要两个迭代器，起始元素，终止元素后面一个元素

copy, remove, fill, replace, random_shuffle, swap

equal, mismatch, find, count, search, count_if, for_each, search_n

大多在<algorithm>中定义

template <class InIt, class T>
InIt find(InIt first, InIt last, const T & val);

first和last都是容器的iterator，给出了容器查找区间的起点和终点

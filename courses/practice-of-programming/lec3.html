复制构造函数

只有一个参数，对同类对象的引用

形如 X::X(X&) 或者 X::X(const X&) 二者选一

如果没有定义复制构造函数，则编译器自动生成

class Complex{
    private:
        double real, imag;
};

int main(){
    Complex c1;
    Complex c2(c1); // 调用缺省的复制构造函数，将c2初始化成和c1一样
}

如果自己定义复制构造函数，则缺省的复制构造函数不存在

复制构造函数的调用：

  1. 用一个对象初始化同类的另一个对象
class Test{
    public:
        Test(){
            std::cout<<"1 "<<std::endl;
        }
        Test(Test & t){
            std::cout<<"2 "<<std::endl;
        }

};

int main(){
    std::cout<<"a: ";
    Test a;
    std::cout<<"b: ";
    Test b = a;
    std::cout<<"c: ";
    Test c;
    c = a;
}

a: 1

b: 2

c: 1

  1. 如果某函数有一个参数是类A的对象，那么该函数被调用时类A的复制构造函数将被调用

  2. 如果函数的返回值是类A的对象时，函数返回时A的复制构造函数会被调用

注：经过优化的编译器可能导致结果不同

注！对象之间用等号赋值不会导致复制构造函数函数被调用！！

class CMyclass{
    public:
        int n;
        CMyclass(){};
        CMyclass(CMyclass & c){n = 2 * c.n;};
};

int main(){
    CMyclass a, b;
    a.n = 5;
    b = a;
    std::cout<<b.n<<std::endl;
    CMyclass c = a;
    std::cout<<c.n;

delete运算符

delete运算可导致析构函数调用

注：若new一个对象数组，则delete释放时应该写 delete []

using namespace std;
class CMyclass {
    public:
        ~CMyclass() { cout << "destructor" << endl; } 
};

CMyclass obj;
CMyclass fun(CMyclass sobj ) { //参数对象消亡也会导致
                                //析构函数被调用 
    return sobj; //函数调用返回时生成临时对象返回
}
int main(){
    obj = fun(obj); //函数调用的返回值 (临时对象) 被用过后,
    return 0;  // 该临时对象析构函数也被调用
}

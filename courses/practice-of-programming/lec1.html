命令行参数

命令行界面 command line interface

通过键盘输入指令，计算机收到指令会执行

Windows界面下：打开cmd

notepad xyz.txt

copy file1.txt file2.txt (复制文件)

为什么可以传入这些参数？

int main(int argc, char * argv[])
// argc: 参数的个数，至少为1
// argv: 指针数组，每个元素都是一个char*类型的指针
// 例子：
int main(int argc, char * argv[]){
    for(int i=0; i<argc; ++i){
        printf("%s\n", argv[i]);
    }
    return 0;
}

将上述代码编译成exe后

命令行输入：sample para1 para2 s.txt 3 4

输出：

sample

para1

para2

s.txt

3

4

main函数第二个参数的类型是char**！

位运算

负整数的表示：设置“符号位”

位运算：对某一个位进行操作

符号 名称 特点 应用
& 按位与 都是1，才为1 将某些位清0或不变
｜ 按位或 都为0，才为0 将某些位置成1
^ 按位亦或 不同才为1 加密解密；不经过临时变量交换值
~ 按位非 取反 /
<< 左移 将a各个二进制位全左移b位得到的值 用作乘法
>> 右移 右移b位，原符号位为1则补1，为0则补0 用作除法 
void test(){
    int n1 = 15;
    short n2 = -15;
    unsigned short n3 = 0xffe0
    char c = 15;
    n1 = n1 >> 2;
    n2 >>= 3;
    n3 >>= 4;
    c >>= 3;
}

问题：n1, n2, n3, c都是什么？

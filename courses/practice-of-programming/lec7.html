虚函数

类的定义中，前面有virtual关键字的成员函数就是虚函数

NOTE: In Java, all class methods are in default “virtual”!

class base {
    virtual int get();
};

int base::get() {
    ...
}

派生类指针可以赋值给基类指针

通过基类指针调用基类和派生类中的同名虚函数时： * 若该指针指向一个基类的对象，则被调用的是基类的虚函数 * 若该指针指向一个派生类的对象，则被调用的是派生类的虚函数

#include <iostream>
using namespace std;

class A {
    public:
        virtual void foo() {
            cout << "A!" << endl;
        }
};

class B: public A {
    public:
        virtual void foo() {
            cout << "B!" << endl;
        }
};

class D: public A {
    public:
        virtual void foo() {
            cout << "D!" << endl;
        }
};

class E: public B {
    public:
        virtual void foo() {
            cout << "E!" << endl;
        }
};

int main() {
    A a;
    B b;
    E e;
    D d;
    A *pa = &a;
    B *pb = &b;
    D *pd = &d;
    E *pe = &e;
    pa->foo();
    pa = pb;
    pa->foo();
    pa = pd;
    pa->foo();
    pa = pe;
    pa->foo();
    return 0;
}

纯虚函数

没有函数体的虚函数

包含纯虚函数的抽象类为“抽象类”

抽象类不能用来创建抽象对象，但是他的指针和引用可以指向由抽象类派生出的对象

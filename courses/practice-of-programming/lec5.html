友元

友元分为友元函数和友元类两种

  • 友元函数：一个类的友元函数可以访问该类的私有成员
  • 友元类：
class CCar;
class CDriver{
    public:
        void ModifyCar(CCar * pCar);
};
class CCar{
    private:
        int price;
    // friend int MostExpensiveCar(CCar cars[], int total);
    friend void CDriver::ModifyCar(CCar * pCar);
};
void CDriver::ModifyCar(CCar * ptr){
    ptr->price += 1000;
}

int main(){
    return 0;
}

（通过编译！）

友元类

如果A是B的友元类，那么A的成员函数可以访问B的私有成员

class CCar{
    private:
        int price;
        friend class CDriver;
};
class CDriver{
    public:
        CCar myCar;
        void modifyCar(){
            myCar.price += 1000;
        }
};

int main(){
    return 0;
}

（通过编译！）

友元类不能传递，不能继承

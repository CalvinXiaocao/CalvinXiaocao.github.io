## 文件结构

- `report.docx` 实验报告
- `index.html` 主文件，包含 html 以及网页相关的 javascript 代码部分
- `draw-word-cloud.py` 生成词云图的脚本文件（使用 Python Wordcloud 进行词云图生成）
- `data/` 数据集，来自 CoTS 数据集以及 Spotify
- `img/` 网页制作的相关图像
- `songs/` 网页制作的音频文件
- `wordclouds/` 生成的词云图，共11张，对应每个专辑
- `css/` 样式表部分
